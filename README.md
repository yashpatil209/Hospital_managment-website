
# Hospital Management Website

Welcome to the Hospital Management Website repository. This project aims to streamline the operations of a hospital by providing a comprehensive web-based platform for managing patients, staff, appointments, medical records, and billing.


[View Demo](https://hospital-managment-website.onrender.com/)

## Features

*  **Patient Management :-**  Register new patients, update patient information, view patient history, and manage appointments.

* **Staff Management :-** Manage doctor and nurse profiles, schedules, and availability.

* **Appointment Scheduling :-** Book, reschedule, or cancel appointments with ease.

* **Medical Records :-** Maintain detailed medical records, including diagnoses, treatments, and prescriptions.

* **Billing System :-** Generate and manage invoices for medical services rendered.
* User Authentication: Secure login system for different user roles (admin, doctor, nurse, patient).

* **Dashboard :-** An intuitive dashboard providing an overview of hospital operations and key metrics.


## Technologies Used

* **Frontend :-** HTML, CSS, JavaScript
* **Backend :-** Node.js, Express.js
* **Database :-** MongoDB 
* **Deployment :-** Github
